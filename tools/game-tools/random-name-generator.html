How to Use
  1. Select the type of name you want to generate (Fantasy, Sci-Fi, etc.)
  2. Choose gender preference for the names
  3. Select desired name length
  4. Enter how many names you want to generate (1-50)
  5. Choose whether to include surnames and titles
  6. Click "Generate Names" to create your list
  7. Copy individual names by clicking on them or use "Copy All" for the entire list

About Robots.txt

Robots.txt is a text file that tells search engine crawlers which pages or files they can or can't access. Common uses include:

  • Preventing duplicate content indexing
  • Protecting sensitive areas of your site
  • Managing server load
  • Controlling search engine crawling
Note: While robots.txt can prevent crawling, it cannot prevent indexing of already crawled pages. Use meta robots tags for that purpose.
